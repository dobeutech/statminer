# Changelog

All notable changes to StatMiner will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.1.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [0.1.0] - 2025-01-01

### Added

- Multi-model chat interface with support for OpenAI, Anthropic, OpenRouter, Grok, and Requesty providers
- Tabbed, quad, and comparison view modes for simultaneous AI responses
- Real-time streaming responses via WebSocket connections
- Neo4j graph database integration for data storage and knowledge graphs
- NextAuth.js authentication with session management
- Zustand-based state management for chat sessions and user preferences
- Data aggregation from public APIs (US Census Bureau)
- Chart.js and D3.js data visualization
- Progressive Web App (PWA) support with offline capabilities
- Zod validation schemas for API request/response validation
- Job queue system for background processing
- Webhook endpoint for external integrations
- Health check API endpoint
- Tailwind CSS styling with Radix UI components
- Framer Motion animations
- Pino logging for production observability
- Jest testing infrastructure with baseline tests
- Replit deployment configuration
- Comprehensive documentation (README, CONTRIBUTING, SECURITY)
